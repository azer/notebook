# Kubernetes

Sources:

* [Bare Metal with K3s](https://blog.alexellis.io/bare-metal-kubernetes-with-k3s/)

## Installation

```bash
$ yay -S arkade-bin # for arch linux users
$ arkade get kubectl
$ arkade get k3sup
$ arkade get terraform
$ arkade get kubectx
```

Update `PATH`:

```
export PATH=$PATH:$HOME/.arkade/bin/
```
